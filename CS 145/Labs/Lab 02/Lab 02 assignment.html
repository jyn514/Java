Lab 02

THE COIN MACHINE!

 

Objective:

 

Upon entering a value, the program determines how many quarters, dimes, nickels, and pennies to that composes said amount.

 

Example Dialog:

 

Enter a whole number from 1 to 99.

The machine will determine a combination of coins.

87

87 cents in coins:

    3 quarters

    1 dime

    0 nickels

    2 pennies

 

Hint:  Here is an algorithm to solve this problem

  1. Read the amount.
  2. Make a copy of the amount.
  3. Find the maximum number of quarters in the amount.  This may be a good time to use that MOD (%) operator.
  4. Subtract the value of the quarters from the amount.
  5. Repeat the last two steps for dimes, nickels, and pennies.
  6. Print the original amount and the quantities of each coin.

 

Lab Report Questions:

  1. What does the “mod” operator do?
  2. In order to determine whether or not a number was even or odd how can the mod operator be used?
