Lab 07

BMR Calculator

 

Objective:

 

Write a program that calculates a person’s BMR to maintain their current weight

 

  • The user inputs their weight in pounds, height in inches, and age in years
  • Also the user must specify whether they are a sexually assigned male or female
  • Also the user must choose their lifestyle being either Sedentary, Somewhat active, Active, Highly active
  • BMR for a female is

 

BMR = 655 + (4.3 x weight in pounds) + (4.7 x height in inches) – (4.7 x age in years)

 

  • BMR for a male is

 

BMR = 66 + (6.3 x weight in pounds) + (12.9 x height in inches) – (6.8 x age in years)

 

  • If the person is sedentary then increase the BMR by 20%.  Somewhat active increase BMR by 30%.  Active increase the BMR by 40%.  Highly active increase by 50%
